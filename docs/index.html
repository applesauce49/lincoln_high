How to use the table:

Goal: Identify letters that map to your remaining value at each step/row.
When you are finished, you will have a remainder of 0.
  1. Find largest value less than encoded number of your word in a row. The letter in that cell is the first letter of the word.
  2. Subtract the value in the cell from your encoded number.
  3. Find largest value less than the result of step two in the next row up the table. This is the second letter of the word.
  4. Subtract the value in the cell from result of step two.
  5. Find largest value less than the result of step two in the next row up the table. This is the next letter of the word.
  6. Repeat steps 4 and 5 until you have a remainder of 0.
 Example:
Encoded value is 80060
  1. Find largest value less than 80060 in the "19683" row: Column 4 = 78732. This is “D”.
  2. Subtract the value in the cell from 80060: This results in 1328.
  3. Find largest value less than 1328 in the “729” row: Column 1 = 729. This is “A”.
  4. Subtract the value in the cell from 1328. This results in 599.
  5. Find largest value less than 599 in the “27” row: Column 22 = 594. This is “V”.
  6. Subtract the value in the cell from 599. This results in 5.
  7. Find the letter that 5 is associated with in the “1” row. This is “E”.
The result is “DAVE”.
