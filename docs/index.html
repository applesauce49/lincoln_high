How to use the table:

Goal: Identify letters that map to your remaining value at each step/row.
When you are finished, you will have a remainder of 0.
  1. Find largest value less than encoded number of your word in a column. The letter in that row is the first letter of the word.
  2. Subtract the value in the cell from your encoded number.
  3. Find largest value less than the result of step two in the next column in the table. This is the second letter of the word.
  4. Subtract the value in the cell from result of step two.
  5. Find largest value less than the result of step two in the next column in the table. This is the next letter of the word.
  6. Repeat steps 4 and 5 until you have a remainder of 0.
 Example:
Encoded value is 80060
  1. Find largest value less than 80060 in 4th column: 78732. This is “D”.
  2. Subtract the value 78732 from 80060: This results in 1328.
  3. Find largest value less than 1328 in 3rd column: 729. This is “A”.
  4. Subtract the value 729 from 1328. This results in 599.
  5. Find largest value less than 599 in the 2nd column: 594. This is “V”.
  6. Subtract the value 594 from 599. This results in 5.
  7. Find the letter that 5 is associated with in the 1st column: This is “E”.
The result is “DAVE”.
